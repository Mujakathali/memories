Put your photos/videos here so they’re deployed with the site.

Use paths like:
- /media/photos/my-photo.jpg
- /media/videos/my-video.mp4

To make your added memories persist on Vercel, create/update `memory_list.json` in this folder.
Use the app’s `Export memories` button to download it, then paste it into this file and commit + push.

Then commit + push, and Vercel will serve them.
